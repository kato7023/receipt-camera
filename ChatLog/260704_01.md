## 技術的メモ
- clasp は GAS ディレクトリ内の `.ts` ファイルを自動的に認識しない（Untracked になる）→ `.js` で統一
- freeeAPIv2 は自作ライブラリ。`new FreeeAPI.Request('エンドポイント名').requestGET()` / `.request___POST(payload, binary)` で API 呼び出し
- GAS ライブラリのメソッドは `Object.keys()` では列挙不可 → 直接アクセスで確認
- `clasp deploy` を引数なしで実行すると毎回新しいデプロイIDが生成される → `-i` で上書き必須
- PowerShell では `&&` が使えない → コマンドを分割実行
- GitHub Pages はプライベートリポジトリでは Pro プラン以上が必要
